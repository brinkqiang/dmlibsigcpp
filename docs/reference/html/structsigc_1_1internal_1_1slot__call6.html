template <class T_functor, class T_return, class T_arg1, class T_arg2, class T_arg3, class T_arg4, class T_arg5, class T_arg6>
struct sigc::internal::slot_call6< T_functor, T_return, T_arg1, T_arg2, T_arg3, T_arg4, T_arg5, T_arg6 >

Abstracts functor execution.

call_it() invokes a functor of type T_functor with a list of parameters whose types are given by the template arguments. address() forms a function pointer from call_it().

The following template arguments are used:

  • T_functor The functor type.
  • T_return The return type of call_it().
  • T_arg1 Argument type used in the definition of call_it().
  • T_arg2 Argument type used in the definition of call_it().
  • T_arg3 Argument type used in the definition of call_it().
  • T_arg4 Argument type used in the definition of call_it().
  • T_arg5 Argument type used in the definition of call_it().
  • T_arg6 Argument type used in the definition of call_it().
